Basic Media Queries

En esta asignación, aplicarás un diseño responsivo utilizando CSS Grid y media queries.

  • La **vista móvil** (por defecto) usará una cuadrícula de una columna con 10px de espacio.
  • La **vista de tableta** (a partir de 772px) usará una cuadrícula de dos columnas, con figuras circulares y la última ocupando ambas columnas.
  • La **vista de pantalla grande** (a partir de 992px) usará una cuadrícula de tres columnas, sin bordes circulares y haciendo que cada tercera figura ocupe dos columnas.
  • Se incluirán media queries para **movimiento reducido** y **esquemas de color oscuro**.
